# Simple Web3 Wallet Activity Logger

This is a beginner-friendly Web3 project.

This project helps you:
- Connect a crypto wallet
- Detect account changes
- Detect network changes
- Log wallet activity on screen

This project uses:
- HTML
- JavaScript
- MetaMask events

## Features
- One-click wallet connect
- Listen for account change
- Listen for network change
- Show activity logs

## Requirements
- MetaMask installed
- Internet connection
- Basic JavaScript knowledge

## How to Run
1. Download this repository
2. Open `index.html` in browser
3. Click "Connect Wallet"
4. Switch account or network in MetaMask
5. See logs update on screen

## File Structure
- index.html -> Main webpage
- app.js -> App logic
- connect.js -> Wallet connect
- events.js -> Event listeners
- ui.js -> Screen updates
- logger.js -> Log manager
- format.js -> Format logs
- validate.js -> Checks
- config.js -> App settings
- utils.js -> Helpers

## Learning Goal
This project helps beginners learn:
- How wallets send events
- How Web3 apps react to changes
- How to track wallet activity
